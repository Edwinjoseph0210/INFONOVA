# News Portal Project

A modern news portal built with Next.js, featuring:
- Real-time news updates
- Multiple news categories
- Responsive design
- Dark/Light theme
- Interactive UI components

## Features
- Latest Headlines
- News Categories
- Editor's Picks
- Breaking News Updates
- Newsletter Subscription

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
